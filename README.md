# Sima

Xubuntu Live-USB:n konffaus yhdessä paketissa. 

## Käyttö

Buuttaa livetikulle ja avaa terminaali. Syötä seuraavat komennot: 

	wget https://raw.githubusercontent.com/Miyune/sima/master/run.sh
	bash run.sh
	 

Prego!
